**项目简介**
- 浮窗五子棋用户脚本，专为 `https://linux.do/*`。
- 默认不唤起，按 Alt+K 唤起/隐藏小窗。
- 支持透明度、锁定位置、拖拽移动、自定义快捷键。
- 兼容 ScriptCat、Tampermonkey、Violentmonkey（zip 包用于 ScriptCat 导入）。

**主要功能**
- 悬浮小窗棋盘：15×15 棋盘，Canvas 渲染，高分屏适配（DPR）。
- 难度可选：简单 / 普通 / 困难 / 专家。
- 先手切换与重开：一键重开，支持“我先手 / AI 先手”切换。
- 设置面板：透明度调节（40%~100%）、锁定位置、快捷键自定义。
- 状态提示：显示对弈状态与回合提示。
- 本地持久化：保存难度、先手、窗口位置、显示状态、热键、透明度、锁定等。

**快捷键**
- 默认：Alt+K（显示/隐藏浮窗）。
- 修改：点击小窗标题栏的“⚙ 设置”，在“快捷键”输入框中输入类似“Alt+K”“Ctrl+Alt+J”的格式后回车保存。
- 冲突处理：若与浏览器/系统快捷键冲突，建议改用包含 Ctrl 的组合。

**安装方式**
- ScriptCat 导入
  - 前往 Releases 下载对应版本的 zip 包，在 ScriptCat 中选择“导入脚本包”。

**使用说明**
- 打开 linux.do 任一页面，按 Alt+K 唤起小窗。
- 标题栏可拖动，点击“—”按钮最小化；再次按 Alt+K 重新唤起。
- 选择难度，点击“我先手”切换先手；点击“重开”开始新局。
- 设置里可调透明度、锁定位置、防误触；锁定后不可拖动，需取消锁定再移动。
